# 数据采集API接口

## 接口列表

### 1. 历史开奖数据接口
```
https://kjapi.com/hall/hallhistory/txffcqiqu/ksffc
```
- **用途**: 获取腾讯分分彩历史开奖数据
- **方法**: GET
- **返回**: 历史开奖记录列表
- **说明**: `txffcqiqu` 表示腾讯分分彩，`ksffc` 可能表示快速分分彩

### 2. 指定日期历史数据接口
```
https://kjapi.com/hallhistoryDetail/txffcqiqu/2025-11-15
```
- **用途**: 获取指定日期的详细开奖历史
- **方法**: GET
- **参数**: 日期格式 `YYYY-MM-DD`（在URL路径中）
- **返回**: 该日期的所有开奖记录
- **示例**: 
  - 2025-11-15: `https://kjapi.com/hallhistoryDetail/txffcqiqu/2025-11-15`
  - 2025-11-16: `https://kjapi.com/hallhistoryDetail/txffcqiqu/2025-11-16`

## 使用注意事项

1. **请求频率**: 建议添加请求延迟（1-2秒），避免频率过高被封IP
2. **请求头**: 可能需要设置 User-Agent、Referer 等请求头
3. **错误处理**: 需要处理网络错误、超时、JSON解析错误等情况
4. **数据格式**: 实际API返回格式可能与文档示例不同，需要根据实际情况调整解析逻辑
5. **数据校验**: 采集后需要对数据进行校验，确保数据完整性和正确性

## 数据采集建议

- **增量采集**: 定期采集最新数据，避免重复采集
- **批量采集**: 使用日期范围接口批量获取历史数据
- **数据存储**: 建议使用数据库存储，便于后续分析和查询
- **定时任务**: 可以设置定时任务自动采集最新数据
